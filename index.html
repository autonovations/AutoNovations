Nuestra Misión

Convertimos curiosidad en ingeniería real

Autonovations nació de una pregunta simple: ¿qué pasa si en vez de enseñar conceptos abstractos, le damos a los estudiantes un dron barato y les decimos "desármalo, entiéndelo y mejóralo"?

El resultado es un currículo STEAM que conecta física, programación, electrónica, diseño y matemáticas en un solo proyecto tangible. Nada de simulaciones. Todo real.

// Algoritmo PID — corazón del dron
float PID::compute(float setpoint, float actual) {
  float error = setpoint - actual;

  // Proporcional
  float P = Kp * error;

  // Integral (anti-windup)
  integral += error * dt;
  integral = constrain(integral, -100, 100);
  float I = Ki * integral;

  // Derivativo
  float D = Kd * (error - prev_error) / dt;
  prev_error = error;

  return P + I + D;  // output → motor PWM
}

Stack tecnológico

🔧
Hardware: ESP32 + PCB propia
Diseñamos nuestra propia placa con KiCad y la fabricamos en JLCPCB. Los estudiantes aprenden electrónica real, no simulada.
📱
App Android en Kotlin
Joystick virtual, telemetría en tiempo real y dashboard de sensores. Comunicación UDP a 100Hz para latencia mínima.
⚙️
Firmware con FreeRTOS
Loop de control PID a 500Hz. Los estudiantes aprenden scheduling, prioridades de tareas y sistemas embebidos profesionales.
🔒
Seguridad y protocolos
Analizamos las vulnerabilidades del dron original y diseñamos protocolos seguros. Ciberseguridad en hardware real.
