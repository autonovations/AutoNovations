Módulo 1: Arquitectura Computacional ROS2

ROS2 no es un sistema operativo tradicional, sino un Middleware que facilita la comunicación entre procesos distribuidos independientes de su lenguaje (C++ o Python).

Grafo de Nodos y Tópicos

La lógica de control de un robot se descompone en pequeñas unidades autónomas llamadas Nodos. Los nodos intercambian información publicando y suscribiéndose a canales de datos nombrados llamados Tópicos.

# Ejemplo básico de Nodo Publicador en Python (ROS2 rclpy) import rclpy from rclpy.node import Node from geometry_msgs.msg import Twist class TeleopNode(Node): def __init__(self): super().__init__('teleop_node') self.publisher_ = self.create_publisher(Twist, '/cmd_vel', 10) # Publica comandos de movimiento a motores...

Módulo 2: Planificación de Trayectorias y Nav2

El stack de navegación Nav2 de ROS2 proporciona los algoritmos necesarios para permitir que un robot se traslade de forma autónoma entre dos puntos de interés evitando obstáculos estáticos y dinámicos.

Planificadores Globales (Global Planners)

El planificador global calcula la ruta óptima libre de colisiones utilizando algoritmos como A* (A-Estrella) o Dijkstra sobre un mapa de coste (Costmap) estático generado por sensores LiDAR.

// ROS2 C++ Plugin para planificadores Nav2 #include "nav2_core/global_planner.hpp" namespace nav2_straightline_planner { class StraightLine : public nav2_core::GlobalPlanner { // Implementa la ruta libre de colisiones... }; }

Simulador de Planificación de Rutas (Nav2)

Haz clic en cualquier celda de la cuadrícula para agregar/quitar obstáculos (bloques rojos). Luego presiona "Calcular Ruta". El planificador Nav2 (simulado mediante BFS) encontrará el camino óptimo (celdas celestes) desde el inicio (verde/robot) hasta la meta (oro/estrella).
